# 🛍️ Customer Shopping Behavior Analysis

An end-to-end Business Analytics project analyzing customer purchasing patterns, loyalty, subscriptions, discounts, and product performance.

Using **Python, PostgreSQL, SQL, and Power BI**, I transformed transactional data into actionable business insights and interactive dashboards.

### 🔑 Key Insights

* Young adults generated the highest revenue.
* Express shipping customers spent more on average.
* Non-subscribers contributed the majority of sales.
* Loyal customers accounted for nearly 80% of the customer base.
* Discount users included many high-value customers.

### 🛠 Tech Stack

Python | PostgreSQL | SQL | Power BI

### 📊 Skills Demonstrated

Data Cleaning • SQL Analytics • Customer Segmentation • Revenue Analysis • Dashboarding • Business Insights

### 🎯 Business Impact

Identified revenue drivers, customer segments, and retention opportunities to support data-driven business decisions.
