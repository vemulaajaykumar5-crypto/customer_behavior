Customer Shopping Behavior Analysis

This project focuses on analyzing customer shopping behavior across multiple product categories to uncover insights related to spending habits, subscription trends, customer loyalty, discounts, and product performance.

Using Python for data preparation, PostgreSQL & SQL for analytics, and Power BI for visualization, the project transforms raw transactional data into actionable business intelligence.

Tech Stack
Python (Pandas, NumPy)
PostgreSQL
SQL
Power BI
Key Features
Data Cleaning & Feature Engineering
Revenue Analysis by Gender & Age Group
Customer Segmentation
Subscription Analysis
Product Performance Analysis
Discount Usage Analysis
Interactive Power BI Dashboard
Business Insights
Young adults generated the highest revenue
Express shipping users showed higher average spending
Non-subscribers contributed the majority of revenue
Loyal customers represented nearly 80% of customers
Discount users included high-value customers
